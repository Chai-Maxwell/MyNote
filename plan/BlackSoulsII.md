## 一、游戏核心机制

### 1. SEN（理智值）系统

| SEN状态 | 效果 |
|---------|------|
| **高SEN（≥0）** | NPC对话正常，敌人较弱，地图正常显示，适合一周目探索 |
| **低SEN（<0）** | NPC对话变化（可能敌意化），敌人增强，隐藏道路/门出现 |

- **调节方式**：击败/监禁狂鸟贾布加布后获得「**疯狂的齿轮**」，可无限次调节SEN值
- **注意**：低SEN是进入里线和解锁隐藏内容的必要条件

### 2. 死亡机制

- 死后进入**活尸状态**：血量上限-20%，无法在BOSS房门口召唤NPC协助
- **解除活尸方式**：
  - 击败任意BOSS
  - 使用道具「**少女之香**」

### 3. 进度（灯火）系统

- 图书馆共**10盏灯火**，打完一个雾门BOSS点亮一盏
- 以下行为也会+1进度：
  - 拉下里约大圣堂拉杆
  - 进入心脏庭院
  - 开局不打兔子BOSS走捷径见爱丽丝
- **开新篝火和打非雾门BOSS不增加进度**
- **⚠️ 10盏灯全亮且狂鸟存活 → 狂鸟唱歌 → 大部分NPC死亡（不可逆）**

### 4. 升级功能

- 死一次即可传送到**图书馆**
- 击败新手BOSS后解锁防火女（白兔诺登）升级
- 每上交一本童话书获得5个免费等级

### 5. 属性上限

- 基础属性上限约为**20000**左右
- DLC3推荐属性**10000+**

## 二、全地图解锁指南

### 核心枢纽：兔子洞 — 所有冒险的起点

**兔子洞**是整个游戏的交通枢纽，击败新手BOSS后从这里出发前往各个区域：

```
                        坠落之间（起始点）
                             │
                       【兔子洞】←─────────── 游戏核心枢纽
               ┌─────┬───┼───┬─────┐
               │     │   │   │     │
               ▼     ▼   ▼   ▼     ▼
             路德  血泪 烟熏 精神  万能
             维希  之池 之森 病栋  钥匙
             市街                 开门
             主线
```

**兔子洞各出口详解**：

| 方向 | 目的地 | 怎么走 |
|------|--------|--------|
| **左上** | **路德维希市街**（主线） | 击败BOSS**猎颅之兽**后继续往上走即可到达 |
| **右上** | **血泪之池** | 右上小房间喝药传送 |
| **红门上方** | **烟熏之森** | 需要**万能钥匙**开门（图书馆白兔诺登处购买） |
| **红门旁边** | **精神病栋** | SEN≤0 时出现隐藏门（无需万能钥匙） |
| **下方** | 坠落之间 | 返回起始区域 |

> ⚠️ 红门附近还有**兔骑士巴纳伊**的支线，SEN≤0 时触发。

### 主要路线

**路线1（推荐一周目）**：
```
兔子洞 → 路德维希市街 → 雾之公园 → 市街上层 → 牛津学院 → 时计塔
```

**路线2（较难）**：
```
兔子洞（电梯）→ 烟熏之森 → 鱼市场 → 屠宰场 → 卡罗尔川 → 牛津学院
```

## 六、DLC内容

### DLC1：深海篇

- **入口**：叹息的海边 → 牡蛎的腐死海 → 深海
- **主要BOSS**：嚣张的牡蛎 → 深海骑士 → 克缇（秘密姬）→ 大工
- **重要奖励**：星辰戒指、安多鲁之剑、拉来耶之杖、深海骑士的锚

### DLC2：混沌迷宫（OLD KING OF THE CHESS）

- **入口**：图书馆地下监牢右侧红门（或库因兰德红色城堡1楼右下）
- **核心**：Roguelike迷宫，深度0~19
- **深度9**固定BOSS：古王（DLC3前置条件）
- **重要奖励**：黑之斩击、皇家红茶、觉醒等技能书

### DLC3：冬之钟篇（进入条件）

1. 通关**G结局**（完成里线，站在小红帽一边）
2. 进度0时去图书馆地下监牢右侧与**柴郡猫**对话
3. 进度≥1后与混沌迷宫的**梅贝尔**对话
4. 前往**卡罗尔川**，击败古王后从右侧小道进入→坐船→抵达DLC3

> ⚠️ 推荐全属性**10000+**，DLC3难度极高

- 进度5前击杀开膛手杰克
- 与玛丽·安多次互动

1. 雪原不要进入雪屋（否则BE）

3. 回雪屋找格尔达→选择「一起去找吧」
4. 完成支线→最终战可召唤格尔达为梦灵

- 汉诺威废站与女仆互动
- 需要公爵夫人给的火车票前往库因兰德

## 十二、强力通关套路

### 1. 怀表枪流
- **核心装备**：杀人鬼霰弹铳 + 兔子怀表
- **原理**：枪铳不消耗行动回合，怀表无限减CD → 无限射击
- **输出**：堆满攻击力约十万伤害

### 2. 黑之斩击流
- **核心技能**：黑之斩击（混沌迷宫深度≥7）
- **原理**：叠攻击力和回合数，黑之斩击必中高倍率
- **优点**：简单无脑，无法暴击但必中

### 3. 汉斯机关枪流
- **核心装备**：汉斯的机关枪 + 白熊戒指（满暴击）
- **原理**：强化满一次五发，叠回合数秒人
