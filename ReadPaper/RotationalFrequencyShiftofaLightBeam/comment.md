# Fig. 4 分析与代码评价

## Fig. 4 在论文中的作用

Fig. 4 出自 Courtial 等 1998 年 PRL 论文 *Rotational Frequency Shift of a Light Beam*。该图是论文的核心物理解释图，作用是**直观展示圆偏振拉盖尔-高斯（LG）光束的横向电场矢量分布，揭示其具有 (l+σ)-重旋转对称性**。

具体而言：
- 论文的核心结论是：旋转频率偏移 Δω = Ω·J = Ω·(l+σ)，即总角动量量子数 J = l+σ 决定了频移大小
- Fig. 4 从**对称性**角度解释了这个结论——电场分布具有 J 重旋转对称性，光束每旋转一圈，电场经历 J 个周期的变化
- 这与平移多普勒效应完全类比：线动量 p = ħk 对应频移 Δω = v·k，角动量 J = l+σ 对应频移 Δω = Ω·J

图中 2×4 排列的子图分别对应不同轨道角动量量子数 l = 1,2,3,4（列）和偏振手性 σ = -1 (LCP), +1 (RCP)（行），每个子图中标注了 J = l+σ 的值。

## 作图原理

### 物理原理

圆偏振 LG 模的横向复电场可写为：
```
E(r,φ) ∝ (r/w0)^|l| · exp(-r²/w0²) · exp(ilφ) · (x̂ + iσŷ)
```

其中 `exp(ilφ)` 是轨道角动量的螺旋相位，`(x̂ + iσŷ)` 是圆偏振基矢。

取 t=0 时刻的实部（物理电场）：
```
Ex = A(r) · cos(lφ)
Ey = A(r) · (-σ) · sin(lφ)
```

当 σ=-1 (LCP)：Ey = +A·sin(lφ)，电场方向角 = +lφ
当 σ=+1 (RCP)：Ey = -A·sin(lφ)，电场方向角 = -lφ

该矢量场的旋转对称性为 **|J| = |l+σ|** 重，其中 J=0 对应全场径向对称（无离散旋转对称性）。

### 作图要素

1. **背景光强**：LG 模的强度剖面 I ∝ (r/w0)^(2|l|) · exp(-2r²/w0²)，呈环形，中心为零
2. **电场矢量箭头**：在环形峰值附近采样，绘制 Ex, Ey 方向
3. **对称轴标记**：用 dash-dot 线标出 J 条对称轴

## 对代码的评价

### 总体判断

程序的输出**在对称性演示上是有效的**——所有子图的箭头图案确实展示了 |l+σ| 重旋转对称性，这正是 Fig. 4 要传达的核心信息。但在电场矢量的物理准确性上存在一个 bug。

### 核心问题：RCP 行的电场公式错误

代码中实际使用的公式（第 69 行）：
```python
Phi_total = (1 + l + sigma) * Phi_sparse
Ex = A * cos(Phi_total)
Ey = A * sin(Phi_total)
```

而注释中写的（也是物理上正确的）公式是：
```python
Ex = A * cos(l*φ)
Ey = A * (-σ) * sin(l*φ)
```

**两者并不等价。** 展开对比：

| 偏振 | 代码公式 | 正确公式 | 是否一致 |
|------|---------|----------|---------|
| LCP (σ=-1) | Ex=cos(lφ), Ey=sin(lφ) | Ex=cos(lφ), Ey=sin(lφ) | ✅ |
| RCP (σ=+1) | Ex=cos((l+2)φ), Ey=sin((l+2)φ) | Ex=cos(lφ), Ey=-sin(lφ) | ❌ |

- 对于 LCP（下行），1+l+(-1) = l，意外地抵消了多余项，结果碰巧正确
- 对于 RCP（上行），1+l+1 = l+2，多了一个因子 2，电场矢量方向与实际物理不符

**为什么对称性仍然对？** 因为 `(cos(J'φ), sin(J'φ))` 型矢量场的旋转对称性是 |J'-1| 重，代码中 J' = 1+J，所以对称性 = |J|，恰好与正确结果相同。这属于“答对但过程错”。

### 具体表现

以 l=1, σ=+1 (J=2) 为例：
- **正确**：Ex=cos(φ), Ey=-sin(φ)，矢量方向 = -φ（每圈反向转 1 圈），具有 2 重对称性
- **代码**：Ex=cos(3φ), Ey=sin(3φ)，矢量方向 = 3φ（每圈同向转 3 圈），具有 2 重对称性
- 两者在 φ=0, π/2, π 等特殊角度的矢量方向碰巧重合，但在一般角度（如 φ=π/8）完全不同

### 修复方案

将第 69-72 行改为：
```python
Ex = A_sparse * np.cos(l * Phi_sparse + phase_offset)
Ey = A_sparse * (-sigma) * np.sin(l * Phi_sparse + phase_offset)
```

删除 `Phi_total` 变量。

### 其他问题

1. **scale=9.5 硬编码**：该值与网格分辨率 20×20 耦合，改变箭头密度后箭头长度会失调。建议用 `A_sparse` 的最大值做自适应缩放
2. **mask 阈值 0.8**：过滤掉了环形内外两侧的大部分箭头，只保留峰值附近。这在物理上是合理的（减少杂乱），但阈值偏高，环形较窄的 l=1 子图中可见箭头偏少
3. **J=0 的对称轴线**：`range(0, 0)` 为空，避免了除以零，处理正确
4. **`np.abs(l)`**：因为只用了 l≥0，`abs` 是冗余的，但不影响结果

### 综合评价

程序的**可视化效果和对称性演示是合格的**，能够传达 Fig. 4 的核心物理图像。主要缺陷在于 RCP 行的电场矢量方向与真实物理电场存在系统性偏差——虽然不影响对称性的展示，但如果论文审稿人或物理同行仔细核对矢量方向，会发现 RCP 行的箭头不正确。修复方案很简单，就是去掉 `Phi_total` 中的 `+1+sigma` 项，改用注释中已写明的正确公式。
